HealthHub Online Pharmacy

HealthHub Online Pharmacy is an interactive health management platform designed to bridge the gap between patients and healthcare providers in a seamless digital environment. The platform enables secure tele-medicine consultations, convenient appointment scheduling, and continuous health monitoring. Patients can consult with licensed doctors from the comfort of their homes, manage prescriptions, and receive personalized health recommendations based on their medical history.

Technologie(s) Used: C#, SQL Server

Smart Parking System

The Smart Parking System is an innovative Oracle-based application developed using Java to optimize urban parking management. This system leverages sensors and real-time data to track parking space availability, reduce congestion, and guide drivers to the nearest available spot through a mobile interface. The solution incorporates automated billing, reservation of parking slots, and an intuitive admin panel for facility managers. By streamlining the parking process, it enhances the user experience, minimizes time spent searching for parking, and contributes to smarter city infrastructure.

Technologie(s) Used: Oracle , Java

Eventify

Eventify is a full-featured event management platform that simplifies and automates the entire lifecycle of organizing events, from initial registration to collecting feedback after the event. The platform offers modules for attendee registration, ticketing, session scheduling, and venue management. Organizers can monitor real-time participation, send notifications, and gather valuable feedback to improve future events. Eventify's robust backend ensures secure data handling, while its responsive design provides an excellent user experience across devices. The system is suitable for both virtual and in-person events, making it highly adaptable.

Technologie(s) Used: Java
